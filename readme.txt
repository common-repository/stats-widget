== Description ==

Widget to display stats on your site to guests such as the # total of pages,
posts ,comments ,  tags and  categories created and the on the site.


== Installation ==

1. Upload the 'stats-widget' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to  Go to Widgets > Stats Widget
